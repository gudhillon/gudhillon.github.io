Necto
IOS

I'm working with a UCSC Alumnus and a few upperclassmen to make a social media app geared towards college students. The app is similar to reddit with additional features like a match system similar to tinder, but is made for only the use of college students. So far, I created sidebar navigation, new post module for any desired channel, and a notification feature for trending posts, messaging, and new events. We are using UIKit programmatically and integrated Google Firebase to implement app functionalities. We are planning to release the app by the end of this year.
